# Rapid Aging Challenges the Equity of Urban Active Mobility Systems

Code and data for: **"Rapid Aging Challenges the Equity of Urban Active Mobility Systems: A Case Study of Shenzhen, China"**

Ke Zhu, Hongwei Lu*, Dongzhe Liang, Tianci Yao, Peipei Tian*

*Communications Sustainability* (under review)

## Overview

This repository contains the analysis code for assessing the impact of population aging on urban facility accessibility in Shenzhen, China. The study integrates WorldPop gridded population data and Points of Interest (POI) on a 200 m hexagonal grid, comparing standard (5 km/h) and age-adapted (2.7 km/h) walking speeds to quantify service gaps for elderly residents from 2020 to 2035.

## Repository Structure

```
shenzhen-aging/
├── notebooks/                  # Analysis notebooks (run in order)
│   ├── 01_data_preparation.ipynb       # Data cleaning and census correction
│   ├── 02_travel_time_analysis.ipynb   # Travel time and accessibility calculation
│   ├── 03_population_forecast.ipynb    # GPR-based population projection (2020-2035)
│   ├── 04_gwr_analysis.ipynb           # Geographically Weighted Regression
│   ├── 05_china_cities.ipynb           # National-level city aging analysis
│   └── 06_figures.ipynb                # Figure generation (Figs. 1-6)
├── src/                        # Reusable Python modules
├── data/                       # Data directory (download from Zenodo)
│   ├── raw/                    # Original input data
│   ├── processed/              # Intermediate outputs (generated by notebooks)
│   └── output/                 # Final results (generated by notebooks)
├── figures/                    # Generated figures
├── environment.yml             # Conda environment specification
└── requirements.txt            # pip dependencies
```

## Setup

### 1. Clone this repository

```bash
git clone https://github.com/kzhu3118/shenzhen-aging.git
cd shenzhen-aging
```

### 2. Download data

Download the `data/` directory from Zenodo and place it in the project root:

[![DOI](https://zenodo.org/badge/DOI/YOUR_DOI_HERE.svg)](https://doi.org/YOUR_DOI_HERE)

After downloading, extract so that `data/raw/` sits at the project root level.

### 3. Create environment

```bash
# Option 1: Conda (recommended)
conda env create -f environment.yml
conda activate shenzhen-aging

# Option 2: pip
pip install -r requirements.txt
```

### 4. Run notebooks

Execute notebooks in numerical order (01 → 06). Each notebook reads from `data/raw/` or `data/processed/` and writes outputs to `data/processed/` or `data/output/`.

## Data Sources

| Dataset | Source | Resolution |
|---|---|---|
| Gridded population | [WorldPop](https://www.worldpop.org/) | 100 m |
| Points of Interest (POI) | [Amap (Gaode)](https://lbs.amap.com/) | Point |
| Age structure | 7th National Population Census of China (2020) | District |
| Walking travel time | Amap Routing API | Origin-Destination |
| Administrative boundaries | National Geomatics Center of China | Polygon |

## Software Dependencies

- Python 3.11
- Key packages: GeoPandas, scikit-learn (Gaussian Process Regression), mgwr (Geographically Weighted Regression), rasterio, matplotlib

Full dependency list is provided in `environment.yml`.
